KPI Dashboard for EcommerceU

Disclaimer: this is a mock-up project using dataset provided by RevoU.

This project focused in making dashboard for EcommerceU. EcommerceU Company is a growing e-commerce platform. This year (2022), EcommerceU Company seeks to change the way company make decisions to becomme more data-driven. Therefore, experts, as the person in charge of this change, tries to guide the Data Analyst team to provide the proper evaluation and advice regarding how data can be used to provide analysis on the decision-making process that occurs in EcommerceU Company.
